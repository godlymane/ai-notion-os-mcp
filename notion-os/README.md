# NotionOS - AI Startup Operating System via Notion MCP

NotionOS is an AI agent that controls your entire Notion workspace using natural language. Built for the Notion MCP Challenge.

## What it does
- Create pages, databases, tasks via chat commands
- Startup OS template: CRM, Tasks, Finance, Content Calendar
- Human-in-the-loop approval for destructive actions
- MCP protocol compliant JSON-RPC server

## Setup
1. Clone this repo
2. cp .env.example .env
3. Add your NOTION_TOKEN and OPENAI_API_KEY
4. pip install -r requirements.txt
5. python notion_os.py
